Data Types in JavaScript

Boolean

Boolean type data has only two values, "true"and"false". It is very handy to use it in conditionals. For example; to run a code block in an "if" statement.

let a = true;
typeof(a);
//returns "boolean"

Number

As we understand from the name, it represents numeric data. However, number should between -(253 -1) and 253 -1. Otherwise it becomes BigInt.

Null

Null is basically means an empty object. The object exists without an identification. For example;

let n = null;
typeof(n);
//returns "object"

Undefined

Undefined means not defined by assigning a value. For example if a variable declared and not assigned to a value, it becomes undefined.


let myVariable;
typeof(myVariable);
// returns "undefined"

BigInt

BigInt is integer. However, a numeric value beyond the limits (253-1) and -(253-1), becomes BigInt. It is used to represent very large numbers. BigInt value is created by appending n to the end of an integer. For example;

const num = 6546548798465131321654897984654n;
typeof(num);
//returns "bigint"
